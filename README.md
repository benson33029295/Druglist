# Pomelo
handy drug and patient information list for TSGH workers
## 已實現功能
1. 從網站抓取病人藥物資料，適應症、禁忌症副作用、開始結束時間和途徑等
2. 將Pandas表格輸出為html
## 待實現功能
1. 抓取病人其餘資料(診斷、問題清單)
2. 抓取病人常用實驗室數據製成表格
3. 非常用實驗室數據單獨顯示  
4. IO製成表格
5. 對先前測過的實驗室數據顯示本次為上升或下降，方便判讀
## 製成手機app可實現功能
1. 擷取影響檢查圖片
2. 使用者自行設定該病人之重要觀察項目，一鍵回傳該病人重要數據。
